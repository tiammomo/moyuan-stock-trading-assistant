# investor搜索技能

## 技能概述

本技能是一个专门用于搜索A股上市公司投资者关系相关内容的搜索引擎。通过调用同花顺问财的财经资讯搜索接口，帮助用户获取上市公司与投资机构、个人投资者等在各类活动中的交流纪要，及时掌握公司最新业务动态和重要事件披露。

## 技能功能

### 1. 投资者关系活动搜索
- 搜索各类投资者关系活动内容
- 涵盖特定对象调研、分析师会议、媒体采访、业绩说明会、新闻发布会、路演活动、现场参观等
- 支持中文关键词搜索

### 2. 查询处理能力
- 自动拆解复杂查询为多个简单查询
- 示例：用户问"最近贵州茅台和五粮液的投资者关系活动"可以拆分为"贵州茅台投资者关系活动"和"五粮液投资者关系活动"两个查询
- 根据查询复杂度决定调用接口的次数

### 3. 数据评估与扩展
- 自动评估搜索结果是否能回答用户问题
- 如有必要，可调用其他技能或工具扩展数据
- 对搜索结果进行质量评估和排序

### 4. 数据处理与返回
- 对搜索结果进行清洗、转换和格式化处理
- 提取关键信息：公司名称、活动类型、日期、标题、摘要、参与机构等
- 将处理后的数据返回给大模型，生成用户友好的回答格式

## 接口信息

### 基础信息
- **Base URL**: `https://openapi.iwencai.com`
- **接口路径**: `/v1/comprehensive/search`
- **请求方式**: POST
- **认证方式**: API Key (通过 `IWENCAI_API_KEY` 环境变量传递)

### 认证要求
API Key需要通过环境变量 `IWENCAI_API_KEY` 传递，不要硬编码在代码中。

### 请求参数
```json
{
  "channels": ["investor"],
  "app_id": "AIME_SKILL",
  "query": "搜索关键词"
}
```

### 响应格式
```json
{
  "data": [
    {
      "title": "文章标题",
      "summary": "文章摘要",
      "url": "文章网址",
      "publish_date": "YYYY-MM-DD HH:MM:SS"
    }
  ]
}
```

## 使用说明

### 1. 环境配置
在使用本技能前，需要设置环境变量：
```bash
export IWENCAI_API_KEY="your_api_key_here"
```

### 2. 调用时机
- 当用户需要搜索上市公司投资者关系活动时
- 当用户需要查询调研记录、业绩说明会内容时
- 当用户需要了解路演活动、现场参观信息时
- 当用户需要获取公司与投资机构的交流纪要时

### 3. 查询示例
- "贵州茅台投资者关系活动"
- "人工智能公司调研记录"
- "新能源行业业绩说明会"
- "科技公司路演活动"
- "医药公司分析师会议"

### 4. 输出要求
- 在回答中必须标注"数据来源于同花顺问财"
- 提供清晰的活动信息摘要
- 包含公司名称、活动类型、日期等关键信息
- 如有必要，提供原文链接

## 技能内部逻辑

### 1. 查询处理流程
1. 接收用户查询
2. 分析查询复杂度，决定是否拆解
3. 生成标准化的API查询参数
4. 调用问财财经资讯搜索接口
5. 处理API响应数据

### 2. 数据扩展逻辑
1. 评估搜索结果是否足够回答用户问题
2. 如不足，考虑调用其他技能或工具
3. 对数据进行补充和验证

### 3. 结果生成流程
1. 清洗和转换API返回数据
2. 提取关键信息字段
3. 生成用户友好的回答格式
4. 添加数据来源声明

## 注意事项

1. **数据来源声明**: 在所有输出中必须明确标注"数据来源于同花顺问财"
2. **API Key安全**: API Key必须通过环境变量传递，不得硬编码
3. **请求频率**: 注意接口的请求频率限制
4. **错误处理**: 实现完善的错误处理机制
5. **数据质量**: 对搜索结果进行质量评估和过滤
