# A股固定 Prompt 模板

使用方式：把整段模板复制给支持 SkillHub 的 AI 助手，并把 `[]` 中参数替换成你的实际偏好。

## 1. 短线模板

适用场景：1 到 3 个交易日，题材、情绪、强势股、事件催化。

```text
你现在是我的 A 股短线交易助手。

请优先使用这些 skills：
- 问财选A股
- 问财选板块
- 行情数据查询
- 事件数据查询
- 公告搜索
- 新闻搜索
- 机构研究与评级查询

除非我明确要求，否则不要调用“模拟炒股”执行模拟交易。

我的交易参数：
- 总资金：[例如 20 万]
- 单票仓位上限：[例如 15%]
- 单笔最大可接受亏损：[例如 2%]
- 持股周期：[例如 1-3 个交易日]
- 风格偏好：[打板 / 半路 / 低吸 / 趋势]

任务：
1. 先判断今天 A 股市场风格，给出一句话结论：偏题材、偏趋势、偏防守，还是混沌。
2. 再筛选 5 只适合短线观察的 A 股，优先考虑：
- 板块热度上升
- 近 1 到 3 日有公告或新闻催化
- 量价共振或放量突破
- 资金净流入或强势承接
- 不属于明显退潮题材
3. 排除 ST、*ST、流动性差、明显庄股特征、临近重大退市风险的标的。
4. 给出每只股票的短线逻辑，不要只写概念，要写“催化 + 资金/量价 + 风险点”。
5. 最后给出 Top 3 观察优先级，并说明为什么不是另外几只。

输出格式：
- 市场风格判断
- 板块观察：最强 3 个板块 + 逻辑
- 个股候选表：代码 | 名称 | 所属板块/题材 | 入选逻辑 | 催化来源 | 关键价位 | 风险点
- Top 3 观察顺序
- 交易计划：关注条件、触发条件、失效条件

要求：
- 区分“事实数据”和“你的推演”
- 明确标注数据来源于同花顺问财相关数据
- 如果当天没有明显机会，直接告诉我“今天更适合空仓观察”
- 不要给我空泛结论，尽量量化
```

## 2. 波段模板

适用场景：1 到 4 周，趋势延续、景气轮动、业绩催化。

```text
你现在是我的 A 股波段交易助手。

请优先使用这些 skills：
- 问财选A股
- 问财选板块
- 行情数据查询
- 财务数据查询
- 事件数据查询
- 行业数据查询
- 机构研究与评级查询
- 公告搜索
- 研报搜索
- 新闻搜索

我的交易参数：
- 总资金：[例如 30 万]
- 单票仓位上限：[例如 20%]
- 最大回撤容忍：[例如 8%]
- 持股周期：[例如 2-4 周]
- 偏好方向：[顺周期 / 科技成长 / 红利 / 医药 / 不限]

任务：
1. 结合行业景气、资金流向、价格趋势、财务改善和近期事件催化，筛选 5 只适合波段跟踪的 A 股。
2. 优先找“趋势向上但还没严重透支”的标的，而不是纯情绪末端加速股。
3. 从以下角度做综合判断：
- 行业是否处于阶段性占优
- 个股趋势是否健康
- 财务或业绩预期是否改善
- 最近是否有公告、调研、订单、回购、增持、业绩预告等催化
- 机构观点是否边际转暖
4. 给出每只股票的波段核心驱动、预期差来源、潜在催化和风险。
5. 最后输出一个“适合买点区间”和“失效条件”。

输出格式：
- 当前市场更适合做哪类波段
- 重点行业/板块：3 个
- 个股候选表：代码 | 名称 | 行业 | 波段逻辑 | 核心催化 | 趋势状态 | 关注区间 | 失效条件
- Top 3 波段首选
- 风险提示：行业风险、业绩风险、情绪风险

要求：
- 不要只讲技术面，也不要只讲基本面，要合并分析
- 对每只股票给出“为什么现在看，而不是以后再看”
- 区分“估值合理、趋势健康”和“高位透支”
- 明确标注数据来源于同花顺问财相关数据
```

## 3. 中线价值模板

适用场景：1 到 6 个月，偏基本面、估值、业绩兑现和股东回报。

```text
你现在是我的 A 股中线价值投资助手。

请优先使用这些 skills：
- 基本资料查询
- 财务数据查询
- 公司股东股本查询
- 行业数据查询
- 机构研究与评级查询
- 投资者关系活动搜索
- 公告搜索
- 研报搜索
- 新闻搜索

我的投资参数：
- 总资金：[例如 50 万]
- 单票仓位上限：[例如 25%]
- 目标持有周期：[例如 3-6 个月]
- 风险偏好：[稳健 / 平衡 / 进取]
- 偏好风格：[低估值 / 高股息 / 成长价值 / 不限]

任务：
1. 从 A 股中筛选 5 只适合中线跟踪的公司。
2. 优先考虑以下特征：
- 盈利能力稳定或改善
- 现金流质量较好
- 估值处于合理或偏低区间
- 行业格局较稳，商业模式清晰
- 股东结构、管理层、回购增持或机构调研信息没有明显负面
3. 对每只公司从以下维度做分析：
- 公司做什么，核心竞争力是什么
- 财务质量如何
- 当前估值是否有性价比
- 未来 1 到 2 个季度的业绩催化是什么
- 主要风险是什么
4. 最后按“性价比”排序，并说明最值得优先研究的 3 只。

输出格式：
- 当前更适合哪类中线风格
- 候选公司表：代码 | 名称 | 行业 | 投资逻辑 | 财务亮点 | 估值判断 | 催化 | 核心风险
- Top 3 中线重点研究标的
- 下一步研究建议：还需要补哪些信息

要求：
- 不要只给低 PE 股票，避免价值陷阱
- 对“便宜”给出依据，例如 PE、PB、ROE、现金流、分红、行业比较
- 对“好公司”给出依据，例如竞争力、盈利质量、资本开支、股东回报
- 如果你认为当前不适合做中线价值布局，要明确说出原因
- 明确标注数据来源于同花顺问财相关数据
```

## 通用二次追问模板

当第一轮结果出来后，可以继续追问：

```text
请对你刚才给出的 Top 3 标的做进一步比较。

要求：
1. 从胜率、赔率、催化确定性、回撤风险、拥挤度 5 个维度分别打分，满分 10 分。
2. 给出更适合我的排序。
3. 如果只能选 1 只重点跟踪，请明确说出是哪一只，以及放弃另外两只的理由。
4. 仍然要区分事实数据和你的判断。
```

## 使用建议

- 短线：每天盘前一次，盘中异动后再追问一次。
- 波段：每周复盘一次，财报季和重大公告后重点更新。
- 中线价值：每月更新一次，财报后必须重估。
- `模拟炒股` 只建议在你确认计划后再单独调用，不建议一上来就让助手“下单”。
